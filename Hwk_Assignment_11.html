7. Create 10 Erdos-Renyi random graphs of node sizes 100, 200, 300,.,1000 with the assumption that on average each node is connected to 10 other nodes. Repeat the previous step 10 times and plot the average path lengths and mean diameters by graph size.

# Create matrix to store average path lengths
paths <- matrix(ncol = 10, nrow = 10)
for (n in seq(100, 1000, 100)) {
  for (i in 1:10) {
    g.er <- erdos.renyi.game(n, p = (10/(n-1)))
    paths[i, n/100] = average.path.length(g.er)
  }
}
